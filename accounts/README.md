# Run locally
## Requirements
- [Go 1.9.1](https://golang.org/dl/)
- [protoc 3](https://github.com/google/protobuf/releases)

## Install (Mac only):
- `brew install go`
- `brew install protobuf`

## Run:
- `./run.sh`
